## Особенности реализации

### Язык программирования (Lisp)
Синтаксис базируется на S-выражениях. Транслятор выполняет парсинг исходного кода, строит дерево (AST) и преобразует его в линейную последовательность команд для аккумуляторного процессора.

### Архитектура ACC
Процессор обладает одним основным регистром — **Аккумулятором**. 

### Память (Harvard)
*   **Instruction Memory**: Память с командами.
*   **Data Memory**: Память данных. Содержит переменные, константы и строки.

### Система прерываний
Механизм прерываний для обработки асинхронных событий.
1. При возникновении прерывания текущее состояние (IP, AC, PS) сохраняется.
2. Процессор переходит по адресу из таблицы векторов прерываний.
3. Команда возврата из прерывания восстанавливает контекст.
